# 写在前面

对于python而言, 使用pip作为包管理器是十分方便的,在网络状况不错时,直接使用pip在线安装所需的第三方包是一个最直接简单的方法.不过,遇到网络状况,如网速等问题时,使用pip离线安装的方法也不失为一个很好的方法.

# 离线安装

在离线安装中,我们就需要搞清楚的一个问题,是使用预编译好的.whl文件还是使用源码从头在本地编译安装.如果是AMD或Intel处理器上的Window系统或者Linux系统,一般都有预编译好的.whl文件, 但是如果在一些特殊的系统,如Nvidia的嵌入式计算机Nano等,可能没有对应的.whl文件,这时就需要从源码编译安装.所以离线安装的原则就是,先找自己系统是否有对应包的.whl文件,如果没有则使用对应包的源码文件完成安装.

### whl文件安装

使用whl文件安装的过程十分简单,命令行如下
```
pip3 install XXX.whl
```
注意: .whl文件必须要符合自己本地的系统,不然会出错

### 源码安装

#### 两步操作
* 首先下载源码压缩文件.zip或其他格式,如numpy-1.18.1.zip
* 解压后运行如下命令
```
pip3 wheel ./numpy-1.18.1
pip3 install numpyXXX.whl
```
注意: 
    1. `./numpy-1.18.1`为源码文件解压后的目录的相对路径,也可以使用绝对路径,运行命令后,会把源码文件编译成.whl,并且保存在当前工作目录下
    2. 容易看出,源码安装需要两步,先编译后安装.

#### 一步操作

```
pip3 install ./numpy-1.18.1
```
